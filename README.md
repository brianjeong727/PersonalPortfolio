# Brian Jeong — Personal Site

Personal portfolio for Brian Jeong, CS + Data Science at the University of Pittsburgh.

**Live:** [brianjeong.dev](https://brianjeong.dev) *(custom domain — pending)*

## Stack

- [Next.js](https://nextjs.org) (App Router) + TypeScript
- Custom canvas animation system (fog, neural, heart, build worlds)
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
